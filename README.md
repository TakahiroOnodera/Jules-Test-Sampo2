# ランダム散歩先提案アプリ

## 概要

現在地周辺の施設（カフェ、レストラン、公園）をランダムに一つ提案し、そこまでの徒歩ルートを地図に表示するWebアプリケーションです。
「どこに行こうか迷った時」に、偶然の出会いを後押しします。

## 機能

- **ランダムな場所の提案**: 現在地周辺にある施設をランダムに一つ提案します。
- **カテゴリ絞り込み**: 「カフェ」「レストラン」「公園」の中から好きなカテゴリを選んで検索対象を絞り込めます。
- **ルート表示**: 提案された場所までの徒歩ルートを地図上に分かりやすく表示します。
- **詳細なログ出力**: 開発や保守の助けとなるように、アプリケーションの動作ログをブラウザのコンソールに詳細に出力します。

## セットアップと起動方法

### 1. Google Maps APIキーの取得

このアプリケーションを動作させるには、Google Maps Platform APIキーが必要です。

1.  **Google Cloud Platform (GCP) プロジェクトの作成**:
    *   [Google Cloud Platformコンソール](https://console.cloud.google.com/)にアクセスし、新しいプロジェクトを作成します（または既存のプロジェクトを選択します）。
    *   GCPプロジェクトで請求先アカウントが有効になっていることを確認してください（Google Maps Platformの利用には請求先の有効化が必須です）。

2.  **APIの有効化**:
    *   作成したプロジェクトのダッシュボードで、以下の3つのAPIを検索し、それぞれ有効にしてください。
        *   **Maps JavaScript API**
        *   **Places API**
        *   **Directions API**

3.  **APIキーの作成**:
    *   「APIとサービス」 > 「認証情報」に移動し、「認証情報を作成」 > 「APIキー」を選択して新しいAPIキーを作成します。
    *   セキュリティのため、作成したAPIキーには適切な制限（HTTPリファラーなど）をかけることを強く推奨します。

### 2. APIキーの設定

取得したAPIキーを、このプロジェクトのJavaScriptファイルに設定します。

1.  プロジェクト内の `js/main.js` ファイルを開きます。
2.  ファイルの先頭にある以下の部分を見つけます。
    ```javascript
    const API_KEY = 'YOUR_GOOGLE_MAPS_API_KEY';
    ```
3.  `'YOUR_GOOGLE_MAPS_API_KEY'` の部分を、先ほど取得したご自身のAPIキーに置き換えてください。

### 3. アプリケーションの起動

1.  プロジェクトのルートディレクトリにある `index.html` ファイルを、お使いのWebブラウザ（Google Chrome, Firefox, Safariなど）で開きます。
2.  ブラウザから位置情報の利用許可を求められますので、「許可」を選択してください。許可しない場合、アプリケーションは動作しません。

## 使用方法

1.  **カテゴリ選択**: 画面上部の「カフェ」「レストラン」「公園」ボタンから、探したい場所のカテゴリを選択します。
2.  **検索実行**: 「近くの場所を探す」ボタンをクリックします。
3.  **結果の確認**:
    *   ボタンの下に、提案された場所の名前と住所が表示されます。
    *   地図上には、現在地からその場所までの徒歩ルートが青い線で描画されます。
