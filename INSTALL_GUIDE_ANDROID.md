# Androidアプリ インストールガイド（USB接続なし）

PCとAndroidデバイスをUSBケーブルで接続できない場合のインストール手順です。
クラウドストレージ（Google Driveなど）を利用する方法が最も簡単で確実です。

## 手順

### 1. PCでの作業
1.  ビルドで生成された`app-debug.apk`ファイルを見つけます。
    *   **場所:** プロジェクトフォルダ内の`android\app\build\outputs\apk\debug\`にあります。
2.  `app-debug.apk`ファイルを、お使いのGoogle DriveやDropboxなどのクラウドストレージにアップロードします。

### 2. Androidデバイスでの作業
1.  スマートフォンで、Google Driveなどのクラウドストレージ用アプリを起動します。
2.  アップロードした`app-debug.apk`ファイルを探し、タップしてデバイスにダウンロードします。
3.  ダウンロード完了後、Android標準の**「ファイル」**アプリ（または「マイファイル」など）を起動します。
4.  「ダウンロード」フォルダを開き、`app-debug.apk`を見つけてタップします。
5.  インストール画面が表示されたら、指示に従ってインストールを完了させます。

---

## 【重要】 事前設定

Androidのセキュリティ機能により、提供元が不明なアプリ（Playストア以外から入手したアプリ）をインストールするには、事前の許可が必要です。

1.  Androidの**「設定」**アプリを開きます。
2.  「アプリ」や「セキュリティ」といった項目に進み、**「不明なアプリのインストール」**（または「提供元不明のアプリ」）という設定項目を探します。
3.  アプリの一覧から、今回APKのダウンロードに使うアプリ（例: **Chromeブラウザ**, **Google Drive**など）を選択します。
4.  **「この提供元のアプリを許可」**のスイッチをオンにします。

この設定を行わないと、APKファイルをタップしてもインストールがブロックされますので、必ずご確認ください。
